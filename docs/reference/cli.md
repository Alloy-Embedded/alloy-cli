# CLI reference

Every `alloy <verb>` rendered automatically from the Click tree
at `alloy_cli.main:cli`.  Adding a new command in `commands/`
surfaces the new verb on the next site build — no doc edits.

For a one-page printable reference see the
[Cheatsheet](../CHEATSHEET.md).  For long-form usage walkthroughs
head to the [User Guide](../user-guide/toolchain.md).
